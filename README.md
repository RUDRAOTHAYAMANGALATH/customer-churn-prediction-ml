# Customer Churn Prediction using Machine Learning

This project predicts whether a telecom customer is likely to leave the service using machine learning techniques.

## Project Overview
Customer churn is a major challenge for telecom companies. This project uses machine learning to analyze customer data and identify patterns that indicate whether a customer may stop using the service.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Model
A Logistic Regression model was trained on the Telco Customer Churn dataset to classify customers as churn or non-churn.

## Project Steps
1. Data preprocessing
2. Handling missing values
3. Feature encoding
4. Model training
5. Model evaluation

## Dataset
Telco Customer Churn dataset from Kaggle.

## Result
The model achieved approximately **80% prediction accuracy** in identifying customers likely to leave the service.
